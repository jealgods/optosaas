## Form Elements

### Text Input
- **OPS Patient Identifier** - Free text field for unique patient ID

### Dropdown Selects (Long Lists)
- **Appointment Type** - 23 options from PRD
- **NHS Reason** - 6 options from PRD  
- **Pre Screener** - List of dispensers in system
- **Optometrist** - List of optometrists in system
- **Appointment Outcome** - 5 options from PRD
- **Did Patient Dispense?** - 7 options from PRD
- **Handover** - List of dispensers in system
- **Dispenser** - List of dispensers in system

### Pill/Button Selects (Short Options)
- **Arrival Status** - 4 options (Arrived, Failed to Attend, Cancelled, Rescheduled)
- **OCT** - 6 options (Yes, No, N/A, Clinical, Free, Staff)
- **Where was OCT Booked?** - 2 options (In Store, By Calls Hub)
- **Did Optom Advise New?** - 2 options (Yes, No)
- **Did Optom Discuss CL's?** - 2 options (Yes, No)
- **Did Px Book a Trial?** - 2 options (Yes, No)

### Form Sections
- **Patient Identification** - Basic patient info
- **Appointment Details** - Type and NHS eligibility
- **Arrival & Screening** - Status and pre-screening
- **Clinical Services** - OCT, optometrist, outcomes
- **Dispensing Process** - Handover and dispensing staff

### Mobile Optimizations
- **Stacked Layout** - Single column for all elements
- **Larger Touch Targets** - Pills and buttons sized for fingers
- **Section Dividers** - Clear visual separation
- **Full-width Buttons** - Save/Cancel buttons span full width
- **Responsive Pills** - Multiple rows for pill selections
